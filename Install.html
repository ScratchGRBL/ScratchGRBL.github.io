How ScratchGRBL work


 ScratchGRBLis an extension whick run on Scratch3.0 based software to control hardware like arduino uno, arduino mega drived by GRBL1.1.

Install ScratchGRBL for kittenblock


1.  You need to download and install Kittenblock on your pc from https://www.kittenbot.cc/pages/software.
2.  After you install Kittenbock. You can open a folder on your computer with path "C:\Users\username\AppData\Roaming\Kittenblock\extensions". The 'username' need to be replaced with your own one.  
3.  Download 's3ext-ScratchGRBL' from github  and copy it into folder "C:\Users\username\AppData\Roaming\Kittenblock\extension".
4.  Open Kittenblock and load extension ScratchGRBL.
5.  Enjoy ScratchGRBL , you also can put issue and  discuss  over it.
